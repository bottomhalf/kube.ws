Remembering his PhD experience, Stroustrup set out to enhance the&nbsp;<a href="https://en.wikipedia.org/wiki/C_(programming_language)" title="C (programming language)" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">C</a>&nbsp;language with&nbsp;<a href="https://en.wikipedia.org/wiki/Simula" title="Simula" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">Simula</a>-like features.<sup id="cite_ref-evolving_18-0" class="reference" style="line-height: 1; unicode-bidi: isolate; white-space: nowrap; font-size: 11.2px;"><a href="https://en.wikipedia.org/wiki/C%2B%2B#cite_note-evolving-18" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">[18]</a></sup>&nbsp;C was chosen because it was general-purpose, fast, portable and widely used. As well as C and Simula's influences, other languages also influenced this new language, including&nbsp;<a href="https://en.wikipedia.org/wiki/ALGOL_68" title="ALGOL 68" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">ALGOL 68</a>,&nbsp;<a href="https://en.wikipedia.org/wiki/Ada_(programming_language)" title="Ada (programming language)" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">Ada</a>,&nbsp;<a href="https://en.wikipedia.org/wiki/CLU_(programming_language)" title="CLU (programming language)" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">CLU</a>&nbsp;and&nbsp;<a href="https://en.wikipedia.org/wiki/ML_(programming_language)" title="ML (programming language)" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">ML</a>.</p><p style="margin: 0.5em 0px; color: rgb(32, 33, 34); font-family: sans-serif; font-size: 14px; background-color: rgb(255, 255, 255);">Initially, Stroustrup's "C with Classes" added features to the C compiler, Cpre, including&nbsp;<a href="https://en.wikipedia.org/wiki/Class_(computer_programming)" title="Class (computer programming)" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">classes</a>,&nbsp;<a href="https://en.wikipedia.org/wiki/Derived_class" class="mw-redirect" title="Derived class" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">derived classes</a>,&nbsp;<a href="https://en.wikipedia.org/wiki/Strong_typing" class="mw-redirect" title="Strong typing" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">strong typing</a>,&nbsp;<a href="https://en.wikipedia.org/wiki/Inlining" class="mw-redirect" title="Inlining" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">inlining</a>&nbsp;and&nbsp;<a href="https://en.wikipedia.org/wiki/Default_argument" title="Default argument" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">default arguments</a>.<sup id="cite_ref-hopl2_19-0" class="reference" style="line-height: 1; unicode-bidi: isolate; white-space: nowrap; font-size: 11.2px;"><a href="https://en.wikipedia.org/wiki/C%2B%2B#cite_note-hopl2-19" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">[19]</a></sup></p><div class="thumb tleft" style="float: left; clear: left; width: auto; background-color: rgb(255, 255, 255); margin: 0.5em 1.4em 1.3em 0px; background: rgb(248, 248, 248); color: rgb(0, 0, 0); border: 1px solid rgb(234, 236, 240); border-radius: 2px; padding: 1px 4px; unicode-bidi: embed;"><span class="o" style="color: rgb(102, 102, 102);">++</span><span class="w" style="color: rgb(187, 187, 187);"></span></code>&nbsp;being the&nbsp;<a href="https://en.wikipedia.org/wiki/Increment_operator" class="mw-redirect" title="Increment operator" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">increment operator</a>&nbsp;in C) after going through several other names. New features were added, including&nbsp;<a href="https://en.wikipedia.org/wiki/Virtual_function" title="Virtual function" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">virtual functions</a>, function name and&nbsp;<a href="https://en.wikipedia.org/wiki/Operator_overloading" title="Operator overloading" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">operator overloading</a>,&nbsp;<a href="https://en.wikipedia.org/wiki/Reference_(C%2B%2B)" title="Reference (C++)" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">references</a>, constants, type-safe free-store memory allocation (new/delete), improved type checking, and BCPL style single-line comments with two forward slashes (<code class="mw-highlight mw-highlight-lang-c++ mw-content-ltr" dir="ltr" style="font-family: monospace, monospace; background: rgb(248, 248, 248); color: rgb(0, 0, 0); border: 1px solid rgb(234, 236, 240); border-radius: 2px; padding: 1px 4px; unicode-bidi: embed;"><span class="c1" style="color: rgb(61, 123, 123); font-style: italic;">//</span></code>). Furthermore, Stroustrup developed a new, standalone compiler for C++,&nbsp;<a href="https://en.wikipedia.org/wiki/Cfront" title="Cfront" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">Cfront</a>.</p><p style="margin: 0.5em 0px; color: rgb(32, 33, 34); font-family: sans-serif; font-size: 14px; background-color: rgb(255, 255, 255);">In 1984, Stroustrup implemented the first stream input/output library. The idea of providing an output operator rather than a named output function was suggested by&nbsp;<a href="https://en.wikipedia.org/wiki/Douglas_McIlroy" title="Douglas McIlroy" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">Doug McIlroy</a><sup id="cite_ref-history-cpp-paper_1-5" class="reference" style="line-height: 1; unicode-bidi: isolate; white-space: nowrap; font-size: 11.2px;"><a href="https://en.wikipedia.org/wiki/C%2B%2B#cite_note-history-cpp-paper-1" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">[1]</a></sup>&nbsp;(who had previously suggested&nbsp;<a href="https://en.wikipedia.org/wiki/Pipeline_(Unix)" title="Pipeline (Unix)" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">Unix pipes</a>).</p><p style="margin: 0.5em 0px; color: rgb(32, 33, 34); font-family: sans-serif; font-size: 14px; background-color: rgb(255, 255, 255);">In 1985, the first edition of&nbsp;<i><a href="https://en.wikipedia.org/wiki/The_C%2B%2B_Programming_Language" title="" style="color: rgb(6, 69, 173); background: none; outline-color: rgb(51, 102, 204);">The C++ Programming Language</a></i>&nbsp;was released, which became the definitive reference for the language, as there was not yet an official standard.<sup id="cite_ref-1st-edition3_20-0" class="reference" style="line-height: 1; unicode-bidi: isolate; white-space: nowrap; font-size: 11.2px;"><a href="https://en.wikipedia.org/wiki/C%2B%2B#cite_note-1st-edition3-20" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">[20]</a></sup>&nbsp;The first commercial implementation of C++ was released in October of the same year.<sup id="cite_ref-invention3_17-1" class="reference" style="line-height: 1; unicode-bidi: isolate; white-space: nowrap; font-size: 11.2px;"><a href="https://en.wikipedia.org/wiki/C%2B%2B#cite_note-invention3-17" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">[17]</a></sup></p><p class="MsoNormal" style="margin-top:6.0pt;margin-right:0cm;margin-bottom:6.0pt;
margin-left:144.0pt;text-align:justify;line-height:150%"><span lang="EN-ZW" style="font-size:10.0pt;line-height:150%;font-family:&quot;Times New Roman&quot;,serif;
color:black"></span></p><p style="margin: 0.5em 0px; color: rgb(32, 33, 34); font-family: sans-serif; font-size: 14px; background-color: rgb(255, 255, 255);">In 1989, C++ 2.0 was released, followed by the updated second edition of&nbsp;<i>The C++ Programming Language</i>&nbsp;in 1991.<sup id="cite_ref-2nd-edition3_21-0" class="reference" style="line-height: 1; unicode-bidi: isolate; white-space: nowrap; font-size: 11.2px;"><a href="https://en.wikipedia.org/wiki/C%2B%2B#cite_note-2nd-edition3-21" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">[21]</a></sup>&nbsp;New features in 2.0 included multiple inheritance, abstract classes, static member functions,&nbsp;<a href="https://en.wikipedia.org/wiki/Const_correctness" class="mw-redirect" title="Const correctness" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">const member functions</a>, and protected members. In 1990,&nbsp;<i>The Annotated C++ Reference Manual</i>&nbsp;was published. This work became the basis for the future standard. Later feature additions included&nbsp;<a href="https://en.wikipedia.org/wiki/Template_(programming)" class="mw-redirect" title="Template (programming)" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">templates</a>,&nbsp;<a href="https://en.wikipedia.org/wiki/Exception_handling" title="Exception handling" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">exceptions</a>,&nbsp;<a href="https://en.wikipedia.org/wiki/Namespaces" class="mw-redirect" title="Namespaces" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">namespaces</a>, new&nbsp;<a href="https://en.wikipedia.org/wiki/Cast_(computer_science)" class="mw-redirect" title="Cast (computer science)" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">casts</a>, and a&nbsp;<a href="https://en.wikipedia.org/wiki/Boolean_datatype" class="mw-redirect" title="Boolean datatype" style="text-decoration-line: none; color: rgb(6, 69, 173); background: none;">Boolean type</a>.</p>
